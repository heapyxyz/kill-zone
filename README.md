# kill-zone
`kill-zone` is a SourceMod plugin which kills players when they walk into a zone.

## Requirements
- SourceMod 1.10 or newer
- [Map Zone Library (fixed)](https://github.com/heapyxyz/mzl-fixed)
- [smlib](https://github.com/bcserv/smlib)
